Установка расширения CRM Tasks (без Chrome Web Store)
======================================================

1. Распакуйте архив в любую папку.
   (Должны появиться файлы: manifest.json, sidepanel.html, папки js/, assets/ и т.д.)

2. Откройте Chrome → chrome://extensions/

3. Включите переключатель «Режим разработчика» (справа вверху).

4. Нажмите «Загрузить распакованное расширение».

5. Укажите папку, куда вы распаковали архив (ту, где лежит manifest.json).

Готово. Иконка расширения появится в панели Chrome; нажмите на неё, чтобы открыть боковую панель с задачами.

Для работы с CRM-задачами в настройках расширения укажите URL вашего API и выполните вход.
